1. Book
Родительские классы: отсутствуют
Дочерние классы: отсутствуют

Используется для представления книги в библиотеке. Содержит основные атрибуты, такие как название, автор и год выпуска.

2. Person (абстрактный класс)
Родительские классы: отсутствуют
Дочерние классы: Employee, Client

Базовый класс для всех людей в системе. Содержит общие атрибуты и данные для сотрудников и клиентов, такие как имя и идентификатор.

3. Employee
Родительские классы: Person
Дочерние классы: отсутствуют

Представляет сотрудника библиотеки. Содержит информацию о сотруднике, такую как должность и обязанности. Наследует общие атрибуты от класса Person.

4. Membership (абстрактный класс)
Родительские классы: отсутствуют
Дочерние классы: RegularMembership, PremiumMembership

Базовый класс для типов членства клиентов в библиотеке. Определяет общие атрибуты для различных типов членства.

5. RegularMembership
Родительские классы: Membership
Дочерние классы: отсутствуют

Определяет обычное членство в библиотеке. Содержит базовые преимущества для клиентов с этим типом членства.

6. PremiumMembership
Родительские классы: Membership
Дочерние классы: отсутствуют

Определяет премиальное членство в библиотеке. Содержит дополнительные преимущества по сравнению с обычным членством.

7. Client
Родительские классы: Person
Дочерние классы: отсутствуют

Представляет клиента библиотеки, который может брать книги. Содержит указатель на объект Membership и список книг, которые клиент взял.

8. Catalog
Родительские классы: отсутствуют
Дочерние классы: отсутствуют

Отвечает за управление коллекцией книг в библиотеке. Хранит список книг и предоставляет методы для поиска и добавления книг.

9. Library
Родительские классы: отсутствуют
Дочерние классы: отсутствуют

Центральный класс для управления библиотекой. Содержит объект Catalog, список сотрудников и клиентов. Управляет всем процессом работы библиотеки.

10. Librarian
Родительские классы: Employee
Дочерние классы: отсутствуют

Представляет библиотекаря. Является специализированным сотрудником библиотеки, который управляет процессом выдачи книг клиентам, возвращает книги и консультирует клиентов.

Наследование:
Employee и Client наследуются от Person.
RegularMembership и PremiumMembership наследуются от Membership.
Librarian наследуется от Employee.
